# Joyce Station Tailors & Dry Cleaners

This is a one-page professional website for Joyce Station Tailors & Dry Cleaners in Vancouver, BC. It presents the business identity, tailoring and dry cleaning services, customer reviews, hours, location, and direct call and directions actions for local customers.

## Technology

- TanStack Start with React
- TanStack Router file-based routes
- Tailwind CSS v4 entrypoint with custom CSS
- Lucide React icons
- Netlify deployment support through the generated TanStack Start Netlify setup

## Local Development

Install dependencies, then run the development server:

```bash
npm install
npm run dev
```

The app is implemented primarily in `src/routes/index.tsx` with global styling in `src/styles.css`.
